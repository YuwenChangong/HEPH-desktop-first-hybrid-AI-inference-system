# heph 隐私政策

最后更新：2026-04-23

本隐私政策用于说明 heph（下称“本软件”或“本平台”）如何收集、使用、存储、共享和保护你的个人信息及相关数据。

## 1. 我们处理哪些信息

根据你的使用方式，平台可能处理以下信息：

### 1.1 账号与身份信息

1. 登录账号标识，例如邮箱地址、第三方 OAuth 账号标识、手机号或用户 ID。
2. 认证过程所需的访问令牌、刷新令牌或会话信息。

### 1.2 设备与客户端信息

1. 设备指纹、本机生成的矿工身份标识、miner_name。
2. 操作系统版本、客户端版本、运行模式。
3. 本机 GPU / VRAM 信息、已安装模型列表、任务兼容档位。

### 1.3 任务与会话信息

1. 任务状态、任务 ID、任务来源、模型名、执行模式。
2. 聊天会话元数据、会话标题、创建时间、更新时间。
3. 执行时长、首 token 时间、成功/失败状态、错误类型、重试状态、取消状态。

### 1.4 内容数据

根据你的实际使用和当前产品配置，平台可能处理：

1. 你发送的 prompt、图片或上下文信息。
2. 模型输出内容、推理状态、流式 delta、最终结果。
3. 为了隐私保护而脱敏后的任务展示信息。

### 1.5 计费与审计信息

1. credit 预留、扣费、退款、奖励、结算记录。
2. 与 task_id 对应的对账日志。

## 2. 我们如何使用这些信息

我们处理上述信息主要用于：

1. 提供聊天、推理、接单、自动接单和任务执行服务。
2. 识别本机矿工身份并进行设备能力匹配。
3. 进行任务调度、状态同步、失败回收、幂等校验和日志排错。
4. 进行 credit 计费、奖励、退款和对账。
5. 进行安全审计、滥用检测、反作弊和风控。
6. 改进软件稳定性、兼容性和用户体验。

## 3. 本地存储与远程存储

### 3.1 本地存储

本软件可能在本机存储以下数据：

1. 语言、主题、聊天页偏好、当前模型等本地设置。
2. 本地聊天历史、副本缓存、任务状态缓存。
3. 本地运行时日志、安装状态、模型状态、会话令牌。
4. 模型文件、运行时组件、临时下载文件和安装文件。

### 3.2 远程存储

平台或其第三方基础设施可能存储以下数据：

1. 用户身份标识与登录信息。
2. 任务元数据与任务状态。
3. 必要的任务执行记录、计费记录、失败记录。
4. 为提供产品功能而需要保留的部分内容数据或其脱敏版本。

## 4. 接单场景下的额外说明

若你启用接单、矿工模式或自动接单，平台可能额外处理：

1. 你的 miner_name、设备指纹映射、显存档位和已安装模型信息。
2. 你的任务处理成功率、失败类型、平均首 token 时间、活跃状态。
3. 你接到的任务的必要元数据。

在接单页，我们会尽量仅暴露脱敏后的任务信息；但为了完成接单、能力匹配、风控与对账，系统内部仍可能处理相关任务标识和必要字段。

## 5. 数据共享与第三方

我们不会随意出售你的个人信息。以下情形除外：

1. 为实现登录、数据库、对象存储、短信、邮件、模型运行时等功能，需要与对应第三方服务共享必要数据。
2. 根据法律法规、监管要求、司法机关或行政机关要求，必须提供相关数据。
3. 为防止欺诈、攻击、滥用、违法违规行为，在必要范围内向安全服务或执法机构提供相关信息。
4. 经你明确授权或同意。

## 6. 数据保护措施

我们会采取合理措施保护你的数据，包括但不限于：

1. 鉴权控制、令牌校验、接口权限边界。
2. 任务详情按用户隔离访问。
3. 接单页默认脱敏展示。
4. 日志分级与必要字段最小化。
5. 对敏感配置、服务密钥和后台接口进行访问限制。

但你仍应理解，任何互联网传输或本地设备环境都无法保证绝对安全。

## 7. 你的权利

在适用法律允许范围内，你可以：

1. 查看、修改你的账号信息和部分本地资料。
2. 删除本地聊天记录或卸载本软件。
3. 停止启用接单或自动接单。
4. 申请注销账号或删除特定远程数据（以平台能力和法律要求为准）。

## 8. 未成年人

若你为未成年人，请在监护人同意并指导下使用本软件。若我们发现未按适用法律要求处理未成年人信息，平台有权限制相关功能或要求补充说明。

## 9. 政策更新

我们可能根据产品变化、合规要求或安全需要更新本隐私政策。更新后版本可在软件内、安装器、项目说明页或其他合理位置公布。

## 10. 联系我们

如你对本隐私政策、数据处理或个人信息保护有疑问，请通过平台后续公布的官方联系方式与我们联系。
