# 参与贡献 HEPH

感谢你愿意为 HEPH 贡献代码或文档。

这个公开仓库是 HEPH 的对外代码库，因此贡献内容应保持在当前仓库边界内。

## 提交 issue 之前

- 先确认问题是否属于公开客户端、桌面壳、前端 UI 或非敏感的本地流程
- 不要提交任何密钥、token、私有日志或生产配置
- 如果问题依赖未公开的基础设施，请描述现象，不要暴露内部凭据或私有端点

## 适合公开贡献的方向

- UI 改进
- 桌面打包改进
- 安装器体验优化
- 文档完善
- 本地开发体验优化
- 非敏感的 miner / client 行为修复
- 不依赖私有基础设施的 bug 修复

## 不属于这个公开仓库的内容

请不要提交需要以下条件的 PR：

- 生产凭据
- 私有部署权限
- 计费密钥
- 仅管理员可见的控制面逻辑
- 未公开的内部运维数据

## 开发规范

- 路径尽量保持可移植和相对化
- 不要硬编码个人机器路径
- 不要提交 `.env`、构建产物、安装包、日志、缓存或打包运行时目录
- 默认假设任何发到客户端的内容都可以被用户查看

## Pull Request 检查清单

提交 PR 前请确认：

1. 修改仍然在公开仓库边界内
2. 没有引入密钥或机器相关信息
3. 如果行为变化，文档已同步更新
4. 目标本地工作流仍然可用

## 安全问题

如果你发现的问题可能影响用户数据或敏感基础设施，不要直接在公开 issue 中贴完整利用方式，请优先私下联系项目维护者。
